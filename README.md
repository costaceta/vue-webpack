# Vue webpack
Basic confinguration to use in Vue projects

## Dependencies

* vue

## Dev dependencies

* @babel/cli
* @babel/core
* @babel/preset-env
* @vue/babel-preset-app
* babel-loader
* vue-loader
* vue-template-compiler
* webpack
* webpack-cli
* webpack-dev-server

## Start development server

run: ```yarn dev```
